# Portfolio Website

This repository contains the source code and deployment files for my personal portfolio website.  
It showcases my projects, certificates, skills, and professional background.

## Features

- Dynamic project and certificate sections (powered by JSON files)
- Responsive design for all devices
- Downloadable resume
- Contact form (integration-ready)
- Deployed via GitHub Pages

## Structure

- `public/` — Static assets (images, JSON data, resume PDF)
- `src/` — React source code and components
- `docs/` — Production build output for GitHub Pages

## Deployment

The site is automatically deployed from the `docs/` folder using GitHub Pages.
